# 🏢 HR Analytics Dashboard — Interactive Power BI Dashboard

## Project Overview
This project focuses on building a fully interactive, single-page Power BI dashboard to analyze Human Resources data for Awesome Chocolates. The dashboard covers headcount, salary analysis, gender breakdown, age distribution, qualification insights, and staff growth trends — empowering HR teams to make data-driven decisions. 📊

## Problem Statement
HR teams often struggle to get a unified, interactive view of their workforce data. Traditional reports lack depth and interactivity, making it difficult to identify trends in salary, headcount, and employee demographics. This project bridges that gap by transforming raw HR data into a visually compelling and professional dashboard.

## Objective
To design a single-page interactive Power BI dashboard that provides a complete analytical view of HR data — from high-level KPIs to granular employee-level insights — using Power Query, Power Pivot, DAX measures, and professional dashboard design principles. 🚀

## Dashboard Structure

### Single Page: HR Analytics Dashboard
A comprehensive single-page dashboard with all key HR metrics and visualizations.

#### KPI Cards (Left Panel):
- **Headcount** — Total number of employees (161)
- **Avg. Salary** — Average employee salary ($54K)
- **Avg. Leave Balance** — Average leave balance across all staff (16.4)
- **LBL Over 20 Days** — Employees with leave balance exceeding 20 days (29)

#### Key Visualizations:

| Visual | Description |
|--------|-------------|
| 📊 What do they do? | Horizontal bar chart — Headcount by Job Title |
| 🍩 Gender Breakdown | Pie chart — 55% Female, 45% Male |
| 📈 Age Distribution | Histogram — Age spread of staff |
| 💰 Salary vs. Qualification | Scatter plot — Salary relationship with education level |
| 📉 Staff Growth Trend | Line chart — Cumulative headcount over time (2018-2023) |

## Getting Started

### Prerequisites
- Install **Power BI Desktop** (free) from [Microsoft](https://powerbi.microsoft.com/desktop)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sufiyan003/HR-Analytics-Dashboard.git
   cd HR-Analytics-Dashboard
   ```
2. Open **HR_Analytics_Dashboard.pbix** in Power BI Desktop
3. If data doesn't load, go to **Transform Data → Data Source Settings** and update the file path to your local CSV
4. Click **Refresh** to reload all visuals
